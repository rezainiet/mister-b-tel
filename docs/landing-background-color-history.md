# Historique couleur de fond landing

Ce mémo conserve la valeur de fond actuellement utilisée sur la landing Mister B ainsi que la nouvelle couleur demandée par l’utilisateur, afin de pouvoir revenir facilement à l’ancienne version si nécessaire.

| Élément | Valeur |
|---|---|
| Couleur de fond actuelle enregistrée | `#d9ffd2` |
| Nouvelle couleur demandée pour le fond | `#1BD51C` |
| Fichier source observé | `client/src/pages/Home.tsx` |
| Classe actuelle relevée | `bg-[#d9ffd2]` |

Si l’utilisateur demande de remettre l’ancienne couleur, la valeur à restaurer est `#d9ffd2`.
